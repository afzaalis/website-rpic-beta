# RPIC Website
Aplikasi Reservasi/Booking PC untuk internet cafe MOD di Bandung  

## Tools  
Aplikasi ini dibangun menggunakan:  
- **Frontend**: Vite + React
- **Backend**: Express.js, PHP Native  
- **Database**: MySQL
